# ForgeTrack - Professional Field Service & Job Tracker

**100% Pure Native Android App** — Kotlin + Jetpack Compose + Material 3

## Tech Stack
- **Kotlin** + **Jetpack Compose** + **Material 3**
- **Clean Architecture + MVVM** with Hilt DI
- **Room Database** for offline-first data
- **DataStore** for preferences
- **CameraX** for photo capture
- **Google Maps SDK** + Geofencing API
- **Coil** for image loading
- **WorkManager** for background sync
- **Canvas API** for charts
- **PdfDocument** for PDF reports
- **SpeechRecognizer** for voice notes

## Features
- 📋 Dashboard with stats, charts, active timer
- 🔧 Job management (create, track, complete)
- ⏱️ Live Job with timer, GPS check-in/out
- 📸 CameraX photo capture
- 🎙️ Voice notes with speech-to-text
- ✍️ Digital signature capture
- 📄 Auto PDF report generation
- 👥 Client management
- 📊 Analytics with charts
- 📜 History with map view
- ⚙️ Settings (themes, export, notifications)

## Build
```bash
./gradlew assembleDebug    # Debug APK
./gradlew bundleRelease      # Release AAB
```
